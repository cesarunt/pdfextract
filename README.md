# ExtractPDF

ExtractPDF is an script that get data page by page in order to consolidate the information in the next format:
- title pdf
- author name
- objective
- metholodogy (design, approach, level)
- sample
- tools
- conslusions
- background

## Instalation
Place 'process.py' in the current working directory

## Usage
'python3 process.py'
